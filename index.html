City of Seattle Dataset Generator

Will generate the data from the City of Seattle: data.seattle.gov.

The process is exemplified using a dataset from the webpages, City of Seattle Webstats and % vehicles sold by dept.

The user can select a dataset from the options. Each has a value that is the URL for the json page for that dataset. On the JavaScript side, I use this URL to get the JSON data from that page. Then I keep track of the important information that shows up on the columns (under the "columns" json array object, any index with an id != -1 is recorded. Then, it iterates through each JSON object in "data", using the important indexes we recorded from "columns" to pull out the information we want from the objects, and output them in a list. This way, you can visualize what each row's important information is fairly quickly, and can switch between different datasets with ease.
